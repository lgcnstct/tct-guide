💻 MONITO 사용 방법

원격 시험은 반드시 MONITO를 통해 진행됩니다.
아래 절차를 처음부터 끝까지 순서대로 따라 진행해 주세요.

STEP 1. 시험 ID / 접속 코드 입력
  • 이름과 회사 이메일을 입력합니다.
  • 메일·알림톡으로 받은 시험 ID / 접속 코드를 입력합니다.
STEP 2. 접속 URL 주의사항
  • monito.io/tests/onsite로 접속해야 합니다.
  • 일반 사이트 접속 시 시험 응시가 불가합니다.
STEP 3. 사전체험 진행
  • 사전체험은 실제 시험과 동일한 절차입니다.
  • 시험 1일 전까지 반드시 진행합니다.
STEP 4. 시험장 입실
  • 시험 시작 50분 전부터 입실 가능합니다.
STEP 5. 모니토 로고 확인
  • 모니토 로고가 보여야 정상 접속 상태입니다.
STEP 6. 시험 규정 확인
  • 시험 규정을 확인 후 다음으로 진행합니다.
STEP 7. 이용 동의
  • 필수 항목만 동의, 선택 항목은 동의하지 않습니다.
STEP 8. 화면 공유 시작
  • 전체 화면 공유만 허용됩니다.
STEP 9. 화면 공유 확인
  • 공유 중 상태 표시를 확인합니다.
STEP 10. 듀얼모니터 사용 시 주의
  • 사용하지 않는 화면은 종이 등으로 가려야 합니다.
STEP 11. 웹캠 영상 확인
  • 본인의 얼굴이 선명하게 보여야 합니다.
STEP 12. 마이크·카메라 권한 허용
  • 권한 요청 시 반드시 허용합니다.
STEP 13. 소리 확인
  • 알림음이 정상적으로 들리는지 확인합니다.
STEP 14. 시험 시작 대기
  • 시험 시작 시간까지 대기합니다.
STEP 15. 시험 시작 클릭
  • 시험 시작 버튼 클릭 시 시험이 시작됩니다.
